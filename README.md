# eggVueSpaTemplate

egg + vue + sequelize + mysql + jwt

### 开发

#### 前端
```bash
$ cd vue
$ npm i
$ npm run serve
$ open http://localhost:8080/
```

#### 后端
```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 部署
#### 前端
> build后会复制到 `app/publish`

```bash
$ npm build
```

#### 后端

```bash
$ npm start
$ npm stop
```
